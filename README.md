# CloudOps Deploy

A zero-config CI/CD deployment platform (UI prototype)

## Run Locally

### Backend

cd backend
npm install
node index.js

### Frontend

cd frontend
npm install
npm run dev

## Features

* GitHub repo input
* Stack detection (Node, Python, Java)
* Deployment simulation
* Modern SaaS UI

## Note

This is a UI + architecture prototype. Real deployment pipeline is not integrated yet.
